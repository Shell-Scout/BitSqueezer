BitSqueezer
This utility is designed to compress texture packs and resource packs.

Instructions
1.Move the executable file (.exe) into the same folder as your texture/resource pack archive.
2.Rename your original pack to InputPack.
3.Run the program.
4.Once the process is complete, you will receive the compressed file named OutputPack.zip.

Additional Information
Errors: If any issues occur, please refer to the errors file for details.
Cache: The cache file is only required during the compression process. You can safely delete it after the process is finished.
